# Excel Generator Skill Package

## 技能定义

Excel文档生成技能包，专门用于将网文改编项目的多个组件（剧情、脚本、分镜、图像提示词）整合到一个结构化的Excel文档中。

## 调用时机

当需要进行以下任务时调用此技能：

### 项目整合输出
- 将剧情拆解、单集剧本、电影分镜、图像提示词整合到Excel
- 创建多工作表的结构化文档
- 便于项目管理和团队协作

### 数据结构化
- 按集数组织数据
- 按镜头编号排列信息
- 创建便于查阅的表格格式

### 批量处理
- 自动读取项目中的所有相关文件
- 批量处理多集内容
- 统一格式输出

## 资源文件调用

### 核心生成方法
- `excel-generation-method.md` - Excel生成核心方法论
- `data-processing.md` - 数据处理和清洗规范
- `table-structure.md` - 表格结构设计规范

### Python实现
- `excel_generator.py` - 主要的Excel生成脚本
- `data_parser.py` - 数据解析工具
- `format_helper.py` - 格式化辅助工具

## 调用方式

```
调用 excel-generator-skill 时需要读取：
1. excel-generation-method.md（了解生成方法）
2. excel_generator.py（执行Python脚本）
3. 项目中的相关文件（剧情、脚本、分镜、提示词）
```

## 输入要求
- 项目根目录路径
- 需要处理的集数范围
- 输出文件名和路径

## 输出格式
- .xlsx格式的Excel文件
- 多个工作表，每个工作表包含不同类型的内容
- 结构化的表格数据，便于查阅和编辑

## 专业能力

### Python编程
- 使用pandas和openpyxl库操作Excel
- 文件读取和数据解析
- 错误处理和异常管理

### 数据整合
- 多源数据的标准化处理
- 表格结构的优化设计
- 数据关联和索引建立

### 项目管理
- 文件组织和命名规范
- 版本控制和备份策略
- 团队协作文档生成

此技能包专注于将复杂的网文改编项目数据整合为易于管理和查阅的Excel文档格式。
