# 摄影技术指南

## 镜头语言基础

### 镜头类型 (Shot Types)

#### Wide Shot (远景) - WS
- **用途**：建立环境，展现空间关系
- **情绪表达**：宏大、孤独、渺小
- **使用场景**：
  - 新场景的建立镜头
  - 表现角色与环境的关系
  - 营造史诗感或孤独感
- **构图要点**：角色占画面比例较小，环境占主导

#### Full Shot (全景) - FS
- **用途**：展现角色全身动作
- **情绪表达**：完整、动感、活力
- **使用场景**：
  - 动作场面
  - 角色登场
  - 完整动作展示
- **构图要点**：角色全身入画，适当留白

#### Medium Shot (中景) - MS
- **用途**：日常对话的主力镜头
- **情绪表达**：亲近、自然、平衡
- **使用场景**：
  - 日常对话
  - 情感交流
  - 叙事推进
- **构图要点**：腰部以上入画，背景适中

#### Close-up (特写) - CU
- **用途**：强烈情绪表达
- **情绪表达**：紧张、激动、亲密
- **使用场景**：
  - 重要对白
  - 情绪爆发
  - 角色反应
- **构图要点**：面部占画面主要部分

#### Extreme Close-up (大特写) - ECU
- **用途**：极致情绪，关键细节
- **情绪表达**：震撼、压迫、精细
- **使用场景**：
  - 眼神细节
  - 重要物品
  - 极端情绪
- **构图要点**：局部特征填满画面

#### Over-the-Shoulder Shot (过肩镜头) - OTS
- **用途**：对话中的视角镜头
- **情绪表达**：代入感、对话氛围
- **使用场景**：
  - 双人对话
  - 观察视角
  - 建立关系
- **构图要点**：前景人物肩膀，背景对话者

## 拍摄角度 (Camera Angles)

### Eye-level (平视)
- **特点**：与角色视线等高
- **情绪效果**：客观、中性、自然
- **使用场景**：日常对话、客观叙述
- **心理暗示**：平等关系，真实感

### Low-angle (仰角)
- **特点**：摄像机位置低于被摄对象
- **情绪效果**：威严、压迫、崇拜
- **使用场景**：权威角色登场、英雄时刻
- **心理暗示**：力量感、支配地位

### High-angle (俯角)
- **特点**：摄像机位置高于被摄对象
- **情绪效果**：脆弱、渺小、悲伤
- **使用场景**：角色困境、失败时刻
- **心理暗示**：无助感、被支配

### 3/4 view (四分之三角度)
- **特点**：介于正面和侧面之间
- **情绪效果**：立体、动感、自然
- **使用场景**：人物特写、动作场面
- **心理暗示**：立体真实感

## 摄像机运动 (Camera Movement)

### Static (静止)
- **运动**：摄像机完全静止
- **情绪效果**：稳定、客观、庄重
- **使用场景**：重要对白、庄严时刻
- **节奏感**：慢节奏，适合深度对话

### Push in (推进)
- **运动**：摄像机向被摄对象推进
- **情绪效果**：紧张加剧、注意力集中
- **使用场景**：情绪升级、揭示真相
- **节奏感**：加速节奏，增强张力

### Pull back (拉远)
- **运动**：摄像机远离被摄对象
- **情绪效果**：震撼、全貌揭示
- **使用场景**：真相大白、环境揭示
- **节奏感**：节奏放缓，增强震撼

### Pan left/right (左右摇摆)
- **运动**：摄像机水平旋转
- **情绪效果**：跟随、连接、扫描
- **使用场景**：跟随角色、环境扫描
- **节奏感**：流畅连贯，适中节奏

### Tilt up/down (上下倾斜)
- **运动**：摄像机垂直旋转
- **情绪效果**：揭示、对比、不稳定
- **使用场景**：身高对比、建筑展示
- **节奏感**：节奏变化，创造对比

### Track (跟踪)
- **运动**：摄像机跟随被摄对象移动
- **情绪效果**：动感强烈、沉浸感
- **使用场景**：行走对话、追逐场面
- **节奏感**：快节奏，强烈代入感

## 焦点控制 (Focus Control)

### 前景焦点
- **效果**：突出前景元素，后景虚化
- **用途**：强调重要道具、建立空间层次
- **情绪**：聚焦感、重要性强调

### 中景焦点
- **效果**：中景清晰，前后景适度虚化
- **用途**：人物对话、主要动作
- **情绪**：平衡感、自然观看体验

### 角色脸部焦点
- **效果**：面部清晰，其他区域虚化
- **用途**：情绪表达、重要对白
- **情绪**：亲密感、情绪强化

### 手部动作焦点
- **效果**：手部动作清晰，其他虚化
- **用途**：重要动作、细节展示
- **情绪**：精确感、动作强调

### 环境焦点
- **效果**：整体环境清晰
- **用途**：环境建立、全貌展示
- **情绪**：客观感、完整信息传达

## 构图美学原理

### 三分法则
- **横向三分**：水平线放在上三分之一或下三分之一处
- **纵向三分**：垂直线放在左三分之一或右三分之一处
- **交点放置**：重要元素放在交点位置
- **视觉效果**：更有活力，避免居中的呆板

### 对称构图
- **水平对称**：左右对称，营造平衡感
- **垂直对称**：上下对称，营造稳定感
- **使用场景**：庄严场面、权威展示
- **情绪效果**：庄重、平衡、权威

### 引导线构图
- **利用线条**：道路、建筑边缘、栏杆等
- **引导视线**：将观众视线引向重要元素
- **空间深度**：创造画面的立体感
- **情绪引导**：控制观众注意力流向

### 色彩搭配原理
- **冷暖对比**：利用色温差异营造氛围
- **主色调统一**：保持画面色彩和谐
- **重点色突出**：用少量对比色突出重点
- **情绪色彩**：不同颜色对应不同情绪基调

## 镜头选择决策树

### 基于场景类型
```
对话场景 → Medium Shot/Close-up + Eye-level + Static/Push in
动作场景 → Full Shot/Wide Shot + 3/4 view + Track/Pan
情感场景 → Close-up/ECU + Eye-level/Low-angle + Push in
环境场景 → Wide Shot + High-angle + Static/Pan
```

### 基于情绪需求
```
紧张升级 → Close-up + Push in + 角色脸部焦点
震撼揭示 → Pull back + High-angle + 环境焦点
威严展示 → Low-angle + Static + 前景焦点
亲密交流 → Medium Shot + Eye-level + 角色脸部焦点
```

### 基于叙事功能
```
建立场景 → Wide Shot + Eye-level + Static + 环境焦点
推进情节 → Medium Shot + 3/4 view + Pan + 中景焦点
情绪高潮 → Close-up + Push in + Low-angle + 角色脸部焦点
细节展示 → ECU + Static + Eye-level + 手部动作焦点
```

这套摄影技术体系为分镜设计提供了专业的理论支撑和实践指导。
