# 分镜质量检查标准

## 质量检查体系概览

### 七维度评估框架
1. **技术规范性** - 基础技术要求
2. **镜头逻辑性** - 分镜逻辑合理性
3. **摄影专业度** - 专业摄影技术
4. **内容准确性** - 原文一致性
5. **描述清晰度** - 执行可行性
6. **视觉表现力** - 艺术表现效果
7. **叙事连贯性** - 故事推进效果

### 评分权重分布
- 技术规范性：20%（基础要求）
- 镜头逻辑性：20%（核心逻辑）
- 摄影专业度：15%（专业技能）
- 内容准确性：15%（忠实原作）
- 描述清晰度：10%（执行可行）
- 视觉表现力：10%（艺术效果）
- 叙事连贯性：10%（叙事功能）

## 详细检查标准

### 1. 技术规范性 (Technical Compliance) - 权重20%

#### JSON语法检查 (25分)
**满分标准：**
- JSON格式完全正确，无语法错误
- 所有括号、引号、逗号正确配对
- 数组和对象结构规范
- 字符编码无问题

**扣分标准：**
- 轻微语法错误：扣5-10分
- 结构性错误：扣15-20分
- 无法解析的格式：扣25分

#### 字段完整性检查 (25分)
**必需字段清单：**
```
scene_id, frame_type, place, time, characters,
action, emotion, dialogues, narration, camera,
character_layout, background_pure
```

**满分标准：**
- 所有必需字段完整存在
- 字段名称拼写正确
- 无多余或错误字段

**扣分标准：**
- 缺少1个字段：扣5分
- 缺少2-3个字段：扣15分
- 缺少4个以上字段：扣25分

#### 数据类型检查 (25分)
**类型要求：**
- `characters`: 必须是数组
- `dialogues`: 必须是数组
- `camera`: 必须是对象
- 其他字段：必须是字符串

**满分标准：**
- 所有字段数据类型正确
- 数组内容格式规范
- 对象结构完整

**扣分标准：**
- 1个类型错误：扣5分
- 2-3个类型错误：扣15分
- 4个以上类型错误：扣25分

#### 编号规范检查 (25分)
**scene_id规范：**
- 格式：`"scene_001"`, `"scene_002"`, ...
- 从001开始连续递增
- 同镜头start/end使用相同编号

**满分标准：**
- 编号格式完全正确
- 连续无跳号
- start/end配对正确

**扣分标准：**
- 格式不规范：扣10分
- 编号跳跃：扣15分
- start/end不匹配：扣25分

### 2. 镜头逻辑性 (Shot Logic) - 权重20%

#### 空间不变原则 (25分)
**检查要点：**
- 同镜头start/end必须在同一地点
- place字段描述完全一致
- 不能有空间跳跃

**满分标准：**
- 所有镜头严格遵循空间不变
- place描述准确一致
- 空间转换合理设置新镜头

**扣分标准：**
- 1处空间跳跃：扣10分
- 2-3处空间跳跃：扣20分
- 4处以上空间跳跃：扣25分

#### 时间不变原则 (25分)
**检查要点：**
- 同镜头start/end时间描述一致
- time字段完全相同
- 不能有时间跨越

**满分标准：**
- 所有镜头时间描述一致
- 时间氛围表达准确
- 时间转换设置新镜头

**扣分标准：**
- 1处时间不一致：扣10分
- 2-3处时间不一致：扣20分
- 4处以上时间不一致：扣25分

#### 动作连贯性 (25分)
**检查要点：**
- start到end动作自然延续
- 不能有动作跳跃
- 符合1-2秒时长逻辑

**满分标准：**
- 所有镜头动作连贯自然
- start/end形成完整动作链
- 动作时长控制合理

**扣分标准：**
- 1处动作跳跃：扣5分
- 2-3处动作跳跃：扣15分
- 4处以上动作跳跃：扣25分

#### 镜头数量合理性 (25分)
**标准范围：5-12个镜头**

**满分标准：**
- 镜头数量在5-12个范围内
- 数量与内容复杂度匹配
- 节奏控制恰当

**扣分标准：**
- 少于5个或多于12个：扣15分
- 数量与内容不匹配：扣10分
- 节奏控制不当：扣5分

### 3. 摄影专业度 (Cinematography) - 权重15%

#### 镜头类型选择 (25分)
**评估标准：**
- shot_type选择符合场景需求
- 与情绪表达目标一致
- 专业术语使用正确

**优秀标准：**
- Wide Shot用于建立环境
- Close-up用于情绪强调
- Medium Shot用于日常对话
- 选择逻辑清晰

#### 拍摄角度设计 (25分)
**评估标准：**
- angle选择符合权力关系表达
- Low-angle表现威严
- High-angle表现脆弱
- Eye-level表现平等

#### 摄像机运动 (25分)
**评估标准：**
- movement选择增强叙事效果
- Push in增强紧张感
- Pull back营造震撼
- Static提供稳定感

#### 焦点控制 (25分)
**评估标准：**
- focus设置突出关键元素
- 与镜头意图一致
- 引导观众注意力
- 支持情绪表达

### 4. 内容准确性 (Content Accuracy) - 权重15%

#### 对白来源准确性 (30分)
**检查要点：**
- dialogues严格来源于direct_dialogues
- 不得自创或修改对白
- 时间点匹配准确
- 无合适对白时使用空数组

**满分标准：**
- 所有对白来源准确
- 时间匹配完美
- 空数组使用恰当

**扣分标准：**
- 1处自创对白：扣15分
- 2处以上自创：扣30分
- 时间不匹配：扣10分

#### 场景描述一致性 (25分)
**检查要点：**
- narration基于scenes_raw
- 可适度改写但不能偏离
- 保持叙述性风格
- 支持视觉表达

#### 人物行为合理性 (25分)
**检查要点：**
- characters行为符合原文设定
- action描述合理可信
- 情绪变化符合逻辑
- 角色一致性保持

#### 情绪表达匹配 (20分)
**检查要点：**
- emotion与剧情逻辑一致
- 情绪变化合理自然
- 支持整体叙事意图
- 与视觉元素协调

### 5. 描述清晰度 (Description Clarity) - 权重10%

#### 动作描述具体性 (30分)
**优秀标准：**
- action描述具体明确
- 避免模糊抽象表达
- 现在时态使用正确
- 一句话概括准确

#### 人物站位可执行性 (30分)
**优秀标准：**
- character_layout描述详细
- 包含距离、位置、朝向
- 便于后续执行
- 符合空间逻辑

#### 背景描述独立性 (25分)
**优秀标准：**
- background_pure完全不提人物
- 环境描述详细完整
- 可独立用于背景生成
- 包含材质、光线、氛围

#### 地点描述准确性 (15分)
**优秀标准：**
- place描述具体准确
- 不使用模糊表达
- 符合空间逻辑
- 便于场景搭建

### 6. 视觉表现力 (Visual Expression) - 权重10%

#### 构图变化丰富性 (30分)
**评估要点：**
- 镜头类型搭配合理
- 构图变化丰富有趣
- 避免单调重复
- 视觉节奏恰当

#### 情绪传达有效性 (30分)
**评估要点：**
- 视觉元素支持情绪表达
- 色彩、光影、构图协调
- 摄影技巧增强情感
- 观众情绪引导明确

#### 节奏控制恰当性 (25分)
**评估要点：**
- 镜头切换节奏合理
- 紧张场面快节奏
- 温情场面慢节奏
- 起伏变化自然

#### 视觉冲击力 (15分)
**评估要点：**
- 关键镜头具有视觉冲击
- 重要时刻突出表现
- 创意表达恰当
- 记忆点设置合理

### 7. 叙事连贯性 (Narrative Continuity) - 权重10%

#### 情节推进有效性 (30分)
**评估要点：**
- 镜头序列推动故事发展
- 关键信息传达清晰
- 叙事逻辑连贯
- 无冗余或跳跃

#### 情绪弧线合理性 (30分)
**评估要点：**
- 情绪变化自然合理
- 高潮设置恰当
- 情感起伏有层次
- 符合叙事需求

#### 信息传达清晰性 (25分)
**评估要点：**
- 重要信息有效传达
- 观众理解无障碍
- 关键细节不遗漏
- 信息密度适中

#### 悬念设置恰当性 (15分)
**评估要点：**
- 悬念和张力设置合理
- 观众好奇心维持
- 不过度或不足
- 支持整体节奏

## 综合评分机制

### 分数计算公式
```
最终得分 = 技术规范性×20% + 镜头逻辑性×20% + 摄影专业度×15% +
         内容准确性×15% + 描述清晰度×10% + 视觉表现力×10% + 叙事连贯性×10%
```

### PASS/FAIL判定
- **PASS标准**：最终得分≥80分，且各维度均≥70分
- **FAIL标准**：最终得分<80分，或任一维度<70分
- **优秀标准**：最终得分≥90分，且各维度均≥85分

### 评级体系
- **S级 (95-100分)**：专业制作级别，无需修改
- **A级 (90-94分)**：优秀水准，轻微调整即可
- **B级 (80-89分)**：合格标准，通过质检
- **C级 (70-79分)**：需要修改，重新提交
- **D级 (60-69分)**：问题较多，大幅修改
- **F级 (<60分)**：不合格，重新制作

这套质量标准确保分镜设计达到专业影视制作要求，为后续工作提供可靠保障。
