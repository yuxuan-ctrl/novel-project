## 一、场景分类模板

### 1.1 对话场景模板

**适用场景**：角色之间有对话、交流、对峙的镜头

**必须包含的要素**：
- 表情描述（眼神、嘴部、眉毛）
- 视线方向（看向哪里、视线交汇）
- 互动细节（手势、距离、姿态）

**标准结构**：
```
【图X参考角色】角色【服装：详细描述】，【位置：画面坐标】，【表情：眼神/嘴部/眉毛】，【视线：看向哪里/是否交汇】，【互动：手势/距离】，三层环境，光影氛围，构图镜头，风格锚定词，质量约束。
```

**表情关键词库**：
- 坚毅：眼神坚定、嘴角紧抿、剑眉微蹙
- 冷傲：眼神轻蔑、嘴角冷笑、眉梢上扬
- 愤怒：双眼圆睁、咬牙切齿、眉头紧锁
- 悲痛：眼眶湿润、嘴角颤抖、眉宇哀伤
- 震惊：瞳孔放大、嘴巴微张、眉毛高挑

### 1.2 动作场景模板

**适用场景**：角色有动作、能量释放、战斗互动的镜头

**必须包含的要素**：
- 精确角度（身体倾斜度、手臂角度、头部角度）
- 动作细节（具体在做什么、涉及哪些部位）
- 能量效果（能量类型、颜色、形态、动态）

**标准结构**：
```
【图X参考角色】角色【服装：详细描述】，【姿态：身体X度倾斜，手臂X度角度】，【动作：具体在做什么】，【能量效果：能量类型/颜色/形态/动态】，【光影：能量如何照亮环境】，三层环境，动态光影描述，构图镜头，风格锚定词，质量约束。
```

**角度描述参考**：
- 身体角度：前倾30度、后仰45度、侧身60度
- 手臂角度：平举180度、上举90度、前伸与肩同高
- 头部角度：仰起45度、低下30度、侧转15度

**能量效果描述**：
- 金色能量：耀眼金色、螺旋上升、光粒子四散
- 黑色魔气：幽暗黑色、翻涌扩散、阴冷气息
- 火焰：赤红烈焰、升腾跳跃、热浪扭曲

### 1.3 环境场景模板

**适用场景**：纯场景、环境展示、氛围营造的镜头

**必须包含的要素**：
- 三层环境（远景1-2句、中景1-2句、近景1句）
- 材质细节（石质/木质/金属/云雾等）
- 光影动态（光源类型、光线效果、阴影层次）

**标准结构**：
```
【背景参考场景】远景描述（天空/远方建筑/天际线），中景描述（主体周围环境/其他元素），近景描述（地面/道具/材质细节）。时间，天气，光源类型，光线效果，材质反应，阴影层次。氛围词。构图镜头，风格锚定词，质量约束。
```

**材质描述词库**：
- 石质：千年寒玉、古老符文、龟裂石板、青石铺就
- 木质：古松参天、树皮开裂、松针青翠、树根盘虬
- 金属：锁链反光、剑刃寒光、金属质感、光泽刺眼
- 云雾：云海翻涌、薄雾流动、云隙光、丁达尔效应

### 1.4 群像场景模板

**适用场景**：三个及以上角色同时出现的镜头

**必须包含的要素**：
- 主次关系（谁是主体、谁是背景）
- 景深控制（主体清晰、背景虚化）
- 站位关系（三角形/一字形/圆形等）

**标准结构**：
```
【图一参考A，图二参考B，图三参考C】角色A【服装+位置+姿态】，角色B【服装+位置+姿态】，角色C【服装+位置+姿态】，【站位关系：描述整体站位形状】，【主次关系：谁是主体谁是背景】，【景深：主体清晰背景虚化】，三层环境，构图镜头，风格锚定词，质量约束。
```

**站位关系描述**：
- 三角形站位：A居中高位，B左下，C右下
- 一字形站位：A、B、C从左到右一字排列
- 环形站位：A、B、C围成半圆，包围中心

## 四、质量检查清单

### 4.1 生成前自检

生成每条提示词前，必须确认以下项目：

- [ ] **开头格式**：以【图X参考角色】或【背景参考场景】开头
- [ ] **角色服装**：详细描述颜色、材质、装饰、状态（破损/血迹等）
- [ ] **人物位置**：使用精确坐标（如"画面左侧黄金分割点"、"画面中心偏下1/3处"）
- [ ] **姿态动作**：包含具体角度（如"前倾30度"、"仰起45度"）
- [ ] **互动关系**：描述角色之间的距离、视线、能量连接
- [ ] **三层环境**：远景（天空/远景建筑）+ 中景（周围环境）+ 近景（地面/道具细节）
- [ ] **光影描述**：描述光源角度、光线性质、材质反应
- [ ] **风格锚定词**：包含至少3个风格锚定词
- [ ] **质量约束1**：包含"角色动作精准比例正常"（有动作时）
- [ ] **质量约束2**：包含"光影统一符合场景氛围"
- [ ] **质量约束3**：包含"画面干净不要文字水印logo"
- [ ] **质量约束4**：包含"不要畸形手指和多余肢体"
- [ ] **镜头描述**：包含镜头类型、画幅比例16:9
- [ ] **长度控制**：140-260中文字符

### 4.2 常见错误预防

| 错误类型 | 预防措施 |
|----------|----------|
| "像X一样"比喻 | 严格禁止，直接描述视觉效果 |
| "参考图保持一致" | 必须详细描述服装、发型、表情 |
| 位置模糊 | 使用精确坐标和距离描述 |
| 环境简略 | 必须包含远景+中景+近景三层 |
| 风格词缺失 | 确保至少3个风格锚定词 |
| 质量约束缺失 | 确保包含所有4条质量约束 |

## 五、快速参考卡

### 5.1 位置描述标准词

```
画面坐标：
- 画面中心 / 画面正中央
- 画面左侧黄金分割点 / 画面右侧黄金分割点
- 画面左下1/3处 / 画面右下1/3处
- 画面上方中央 / 画面下方中央
- 画面左上 / 画面右上 / 画面左下 / 画面右下

高度关系：
- 居于高位 / 位于高位
- 处于低位 / 跪在下方
- 高度差明显 / 平视

距离描述：
- 相距约X米
- 中间隔着X
```

### 5.2 镜头类型参考

```
- 极限特写：聚焦面部表情，浅景深
- 特写：肩部以上，突出细节
- 中景：半身或全身，展现动作
- 全景：全身+环境，展现关系
- 远景：环境为主，人物点缀
- 电影级远景：宏大场景，深景深
```

### 5.3 标准提示词模板

**单角色对话场景**：
```

```

**双角色对峙场景**：

```

**纯环境场景**：
```
