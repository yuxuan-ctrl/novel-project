### 问题1：角色特征描述缺失

#### ❌ 错误示例
```json
{
  "shot_number": "bad_001",
  "prompt": "【图一参考林玄】林玄跪在台上，参考图保持一致",
  "has_character": true,
  "characters": ["林玄"],
  "character_refs": ["linxuan.png"]
}
```

**问题分析**：

## 第三部分：快速参考卡

### 必须检查项（生成前自检）

```markdown
□ 【图X参考角色】开头
□ 每个角色详细服装描述（颜色、材质、装饰、状态）
□ 精确人物位置（画面坐标，如"画面中心偏下1/3处"）
□ 具体姿态动作（角度、细节、互动）
□ 三层环境描述（远景、中景、近景）
□ "角色动作精准比例正常"（有动作时）
□ "光影统一符合场景氛围"
□ "画面干净不要文字水印logo"
□ 至少3个风格锚定词（仙侠动漫/国风/线条流畅等）
□ JSON格式正确
```

### 常用风格锚定词

```markdown
【必选至少3个】：
- 仙侠动漫风格
- 中国玄幻国风动漫风格
- 色彩鲜明对比强烈
- 流畅线条
- 戏剧性
- 悲壮感
- 威严
- 压迫感
```

### 镜头类型参考

```markdown
- 极限特写：聚焦面部表情，浅景深
- 特写：肩部以上，突出细节
- 中景：半身或全身，展现动作
- 全景：全身+环境，展现关系
- 远景：环境为主，人物点缀
- 电影级远景：宏大场景，深景深
```
