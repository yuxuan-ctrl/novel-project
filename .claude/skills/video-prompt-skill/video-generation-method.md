# 图生视频提示词生成方法论

## 核心理念

**图生视频（I2V）的本质**：在保持静态图像基础视觉信息的前提下，添加合理的时间维度变化。

## 六要素框架（电影美学增强版）

### 1. 基础画面（继承+电影美学增强）

**来源**：完整继承图像提示词（ai_prompt）+ 电影美学控制元素

**必须包含的电影美学元素**：
1. **光线控制**：光源类型（日光、火光、阴天光等）+ 光线特性（柔光、硬光、侧光、边缘光等）
2. **景别明确**：特写/近景/中景/全景/远景（与shot_type对应）
3. **构图方式**：中心构图、左/右侧重构图、对称构图等
4. **色调氛围**：暖色调/冷色调/混合色调 + 饱和度控制
5. **镜头焦段**：长焦/中焦/广角/超广角（根据景别选择）
6. **机位角度**：平拍/仰拍/俯拍/过肩/航拍

**增强后的示例**：
```
特写镜头，长焦，平拍，中心构图，边缘光，柔光，日光，低饱和度，冷色调，
英俊成熟男子叶凡，深邃眼神如吴彦祖般锐利，文雅气质似胡歌般温润，
白色道袍，山洞深处，昏暗光线从洞口渗入，神秘氛围，中国玄幻风格
```

**电影美学控制清单**：
- ✅ 光源：日光
- ✅ 光线特性：边缘光、柔光
- ✅ 景别：特写镜头
- ✅ 焦段：长焦
- ✅ 机位：平拍
- ✅ 构图：中心构图
- ✅ 色调：冷色调
- ✅ 饱和度：低饱和度

### 2. 相机运动（核心）

**设计原则**：
- 必须与分镜的 `camera_movement` 字段对应
- 描述具体的运动距离、速度、方向
- 强调平滑、匀速、专业感

**标准描述模板**：

| 运动类型 | 标准描述 | 适用场景 |
|---------|---------|---------|
| **Push in** | 镜头从X米匀速推进至Y米，焦点从整体逐渐收紧至主体特写 | 情绪强化、细节展示 |
| **Pull back** | 镜头从X米匀速后退至Y米，视野从局部逐渐扩展至环境全景 | 空间展现、场景交代 |
| **Track** | 镜头侧向/前向跟随角色移动X米，保持相对位置和距离不变 | 动作跟随、行走场景 |
| **Tilt** | 镜头保持位置，缓慢上/下倾斜X度，展现垂直空间层次 | 高度对比、建筑场景 |
| **Static** | 镜头完全固定，画面内仅元素微妙变化，无位移抖动 | 对话场景、细节观察 |
| **Rotate** | 镜头围绕主体顺/逆时针旋转X度，360度展现全貌 | 角色展示、震撼效果 |

**距离参考**：
- Close-up: 推进/拉远 0.5-1米
- Medium Shot: 推进/拉远 1-2米
- Full Shot: 推进/拉远 2-4米
- Wide Shot: 推进/拉远 3-8米

### 3. 角色动作（主体）

**设计层级**：

**微妙动作**（适用于Static镜头或3秒以下）：
- 眼神变化：眼睑抬起、眼球转动、聚焦变化
- 呼吸起伏：胸腔微妙扩张收缩
- 表情微动：嘴角上扬、眉头轻蹙
- 头发/衣物：微风轻拂

**常规动作**（适用于3-5秒镜头）：
- 肢体动作：转头、抬手、站起、走动
- 对话动作：嘴唇开合、说话手势
- 情绪表达：微笑、皱眉、惊讶

**复杂动作**（适用于5秒以上）：
- 连续动作：奔跑→跳跃→落地
- 战斗动作：出拳→闪避→反击
- 特效展示：释放灵力→扩散→消散

**时间轴分段**（重要）：

对于4秒以上的镜头，必须分段描述：

```
【角色动作】
0-2秒：起始动作（建立初始状态）
2-4秒：主要动作（核心表现内容）
4-5秒：结束动作（为下一镜头过渡）
```

示例：
```
0-2秒：苏灵溪从静止状态突然起步，双腿开始快速交替；
2-4秒：全速奔跑，长发和破碎衣裙在风中剧烈飘动；
4-5秒：回头张望，脚步略微踉跄
```

### 4. 环境变化（氛围）

**光线变化**：
- 明暗过渡：从昏暗到明亮 / 从明亮到昏暗
- 色温变化：暖金色→冷蓝色
- 光线移动：阳光斑驳移动、光束扫过

**粒子效果**（玄幻题材特有）：
- 金色灵力：波纹扩散、粒子飘散、光晕闪烁
- 身体消散：化作金色粒子、如沙尘般飘散
- 能量爆发：光环层层扩散、能量波动

**自然元素**：
- 微风：树叶摇曳、衣物飘动、头发拂动
- 尘埃：空气中细小颗粒缓慢飘浮
- 落叶：地面落叶被踩踏扬起

**场景互动**：
- 脚步声响：地面轻微震动
- 物品移动：茶杯水面波纹
- 墙壁反光：光线在石壁上流动

### 5. 时长控制（精确）

**必须明确标注**：
```
【时长】X秒
```

**时长与运动强度对应**：

| 时长 | 运动强度 | 适用内容 |
|------|---------|---------|
| 2-3秒 | 微妙变化 | 呼吸、眨眼、表情微动 |
| 3-5秒 | 常规运动 | 转身、对话、行走 |
| 5-7秒 | 复杂动作 | 奔跑、战斗、多段动作 |
| 7秒+ | 叙事序列 | 完整事件、多角色互动 |

**节奏控制关键词**：
- 缓慢（slow, gentle, gradually）
- 匀速（steady, smooth, consistent）
- 快速（rapid, swift, dynamic）

### 6. 质感要求（标准）

**固定后缀模板**：
```
电影级质感，流畅自然运动，无抖动无跳帧
```

**根据场景类型调整**：

- **玄幻场景**：电影级慢镜头，灵力特效自然流畅，光影变化细腻
- **动作场景**：高速运动镜头，轻微运动模糊增强速度感，动态捕捉流畅
- **对话场景**：稳定镜头，面部细节清晰，呼吸微妙可见
- **环境场景**：宁静安详氛围，光线色温自然过渡，空间纵深清晰

## 完整提示词结构

**标准格式**：
```
【基础画面】（继承ai_prompt）｜【相机运动】具体描述｜【角色动作】分段描述｜【环境变化】细节描述｜【时长】X秒｜【质感】标准要求
```

**字符长度建议**：
- 总长度：200-400字
- 基础画面：80-120字（继承）
- 相机运动：30-50字
- 角色动作：50-100字
- 环境变化：30-60字
- 质感要求：20-30字

## 特殊场景处理

### 特效密集场景（如：灵力爆发）

重点描述特效的时间轴变化：

```
【角色动作】0-1秒：叶凡手指缓慢抬起；1-2秒：指尖涌现金色光点；
2-4秒：金色灵力从指尖爆发扩散，形成波纹状能量场；4-5秒：能量波扩散
至全屏，光芒达到最强
```

### 多角色场景（如：追逐）

明确每个角色的动作：

```
【角色动作】0-2秒：苏灵溪从左侧冲入画面，三名恶徒紧随其后；
2-4秒：苏灵溪回头张望，恶徒加速靠近；4-5秒：距离缩短至5米，
恶徒脸上狞笑清晰可见
```

### 粒子消散场景（如：身体化作粒子）

精确描述消散过程：

```
【角色动作】0-1秒：三人身体轮廓开始模糊；1-3秒：从头到脚逐渐化作
金色微小粒子；3-5秒：粒子如沙尘般向上飘散，身体轮廓完全消失，
空气中仅剩金色余韵
```

## 避免的错误

❌ **错误1：运动描述模糊**
```
镜头推进 ❌
镜头从2米匀速推进至0.5米，聚焦面部 ✅
```

❌ **错误2：动作突兀**
```
瞬间转身 ❌
头部缓慢向右转动90度，用时2秒 ✅
```

❌ **错误3：缺少时间轴**
```
角色奔跑、回头、踉跄 ❌
0-2秒：起步加速；2-4秒：全速奔跑；4-5秒：回头踉跄 ✅
```

❌ **错误4：忽略环境**
```
角色走动 ❌
角色走动，脚步踩在落叶上发出沙沙声，阳光斑驳移动 ✅
```

❌ **错误5：角色描述不一致**
```
男子转身 ❌
英俊成熟男子叶凡，深邃眼神如吴彦祖般锐利，转身 ✅
```

## 工具适配建议

### RunwayML Gen-2
- 支持复杂运动描述
- 推荐时长：3-5秒
- 提示词长度：200-400字

### Pika Labs 1.0
- 适合简洁提示词
- 推荐时长：3秒
- 提示词长度：100-200字（可简化环境描述）

### Stable Video Diffusion
- 更依赖图像本身
- 推荐时长：2-4秒
- 提示词可选（主要控制运动强度）
