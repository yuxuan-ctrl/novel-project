# 网文改编漫剧技能包 (Webtoon-Skill)

## 技能描述

这是一个专业的网文改编漫剧技能包，为 Claude 提供完整的小说到漫剧的转化能力。包含改编方法论、压缩策略、视觉化写作风格、专业剧本创作方法、文档模板和改编示例。

## 技能功能

### 核心能力
1. **冲突识别与提取**：从小说文本中准确识别核心冲突点
2. **情绪钩子分析**：识别悬念、反转、爽点等情绪触发器
3. **压缩策略应用**：删减铺垫，突出戏剧冲突
4. **视觉化转换**：将文字描述转化为视觉符号语言
5. **分集节奏控制**：合理安排每集内容和悬念点
6. **专业剧本创作**：运用三幕式结构和悬念设置技巧

### 调用时机
- **直接从小说生成剧本**（推荐模式）：
  - 读取 adapt-method.md + output-style.md + enhanced-script-method.md
  - 直接基于小说章节生成视觉化剧本
  - 适用于逐章处理流程

- **传统剧情拆解模式**：
  - 读取 adapt-method.md + plot-breakdown-template.md + plot-breakdown-example.md
  - 适用于多章节批量处理

- **单集剧本阶段**：
  - 读取 adapt-method.md + output-style.md + script-template.md + script-example.md
  - 基于剧情点创作单集剧本

## 执行流程

### 直接剧本生成流程（推荐）
1. 读取小说章节内容
2. 读取改编方法论，理解冲突提取标准
3. 读取视觉化写作风格，掌握符号系统
4. 读取专业剧本创作方法（enhanced-script-method.md）
5. 直接基于小说内容生成视觉化剧本
6. 确保每集以【卡黑】结尾

### 传统剧情拆解流程
1. 读取改编方法论，理解冲突识别标准
2. 应用拆解模板，确保格式统一
3. 参考示例，学习标准输出
4. 批次处理小说章节（每批6章）
5. 输出标准化剧情点列表

### 单集剧本流程
1. 读取写作风格要求，掌握视觉化规范
2. 应用剧本模板，确保结构完整
3. 参考示例，理解风格标准
4. 基于剧情点创作单集剧本
5. 确保每集以【卡黑】结尾

## 使用方式

### 方式一：逐章处理（推荐）
使用 `/chapter-flow` 或 `/fullflow-anime` 命令：
- 每次只处理一个章节
- 直接从小说生成剧本
- 避免上下文混乱
- 确保内容准确对应

### 方式二：传统批处理
- 先进行剧情拆解
- 再基于剧情点生成剧本
- 适用于批量处理

## 质量保障

### 输出标准
1. **格式规范**：严格遵循模板结构
2. **内容准确**：忠实还原小说核心冲突
3. **风格一致**：保持视觉化、快节奏风格
4. **逻辑清晰**：确保剧情发展合理
5. **悬念有效**：每集结尾设置强烈期待

### 检查维度
- 冲突强度是否足够
- 情绪钩子识别是否准确
- 压缩策略是否有效
- 视觉描述是否生动
- 节奏控制是否合理
- 三幕式结构是否清晰
- 【卡黑】结尾是否成功

## 输出格式

使用V-符号系统：
- `※` - 场景标记
- `△` - 视觉描述
- `【】` - 对话/特效/音效
- `【卡黑】` - 结尾标记

## 与其他技能的配合

### 与 image-prompt-skill 配合
- 生成的剧本可以直接输入给图像提示词生成器
- 视觉化描述便于转换为图像提示词

### 与 video-generation-skill 配合
- 剧本中的动作描述可直接用于视频生成
- 节奏控制有助于视频时长规划

## 使用说明

该技能包支持两种工作模式：

1. **逐章处理模式**（新）：
   - 直接读取小说章节
   - 生成视觉化剧本
   - 适合现代流水线作业

2. **传统拆解模式**：
   - 先拆解剧情点
   - 再生成剧本
   - 适合深度分析

技能包的设计理念是让 AI 能够灵活选择最适合的改编方式，既能进行深度分析，又能快速产出，生成符合漫剧规格的高质量剧本。
