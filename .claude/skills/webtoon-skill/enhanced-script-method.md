# 增强剧本创作方法论

## 专业剧本创作总监方法集成

### 三幕式结构强化

#### 第一幕（0-25%）
- **开场钩子**（3-5秒必须吸引注意）
  - 用强烈视觉冲击开场
  - 立即引入核心冲突
  - 快速交代必要背景

- **快速引入困境**
  - 展现角色面临的危机
  - 建立情绪基调
  - 预示后续发展

#### 第二幕（25-75%）
- **尝试解决问题**
  - 角色的初步行动
  - 遭遇初步阻碍
  - 冲突开始升级

- **冲突持续升级**
  - 阻碍越来越大
  - 赌注越来越高
  - 推向高潮

#### 第三幕（75-100%）
- **达到高潮**
  - 最激烈的冲突
  - 关键转折点
  - 情绪顶点

- **设置悬念结尾**
  - 在关键时刻切断
  - 强制使用【卡黑】
  - 确保下集吸引力

### 视觉化升级技巧

#### 情感外化原则
```
错误示范：
林玄很愤怒

正确示范：
△林玄双拳紧握，指节发白
△他猛地一拳击向墙壁
【墙壁】：咚！出现裂纹
```

#### 环境烘托技巧
- 用天气映射情绪
- 用光影强化氛围
- 用环境细节暗示心理

#### 动作描写强化
1. **动词选择**：使用强力具象动词
2. **力度描述**：强调速度、力量、幅度
3. **结果展现**：描述动作产生的直接后果

### 角色声音塑造

#### 个性化对话设计
1. 提取3个核心特质
2. 设计专属口头禅
3. 体现社会背景
4. 保持语言风格一致

#### 对话优化原则
- 每句都要推动剧情
- 15字以内为宜
- 用动作辅助表达
- 保留20%潜台词

### 节奏控制公式

- **信息密度**：每分钟2-3个关键信息
- **场景切换**：2-3分钟一次
- **对话频率**：2-4句/场景
- **动作节奏**：3-5秒/镜头

### 悬念设置专业方法

#### 悬念类型
1. **结果悬念**：行动结果未知
2. **身份悬念**：真实身份待揭晓
3. **危机悬念**：危险即将来临
4. **反转悬念**：即将发生巨变

#### 强化技巧
- 视觉冲击强化记忆
- 音效配合增强震撼
- 慢镜头定格关键画面
- 留下象征性物品

### 质量检查清单

#### 结构检查
- [ ] 开场5秒内抓住注意力？
- [ ] 三幕式结构清晰？
- [ ] 节奏紧凑无拖沓？
- [ ] 【卡黑】结尾设置成功？

#### 视觉化检查
- [ ] 所有描述可见可感？
- [ ] 符号系统使用正确？
- [ ] 动作描写具体生动？
- [ ] 场景转换流畅自然？

#### 角色检查
- [ ] 对话符合人设？
- [ ] 性格特质一致？
- [ ] 语言风格独特？
- [ ] 情感表达真实？

### 与webtoon-skill的集成点

1. **调用时机**：在读取script-template.md之后
2. **增强内容**：
   - 三幕式结构分析
   - 节奏控制说明
   - 角色表演指南
   - 悬念设置技巧

3. **输出整合**：
   - 保留webtoon-skill的规范格式
   - 加入专业结构标注
   - 补充表演和镜头建议

### 执行示例

在webtoon-skill生成基础剧本后，应用增强方法：

```
# 基础剧本（webtoon-skill生成）
※斩仙台，寒风怒号
△林玄跪在台上，浑身是血
【慧觉】：林玄，你可知罪？

# 增强后剧本
※斩仙台，诛仙寒风，如刀割骨（第一幕开场）
△高耸入云的玉石高台，寒风呼啸（环境烘托）
△林玄跪在台中央，玄铁锁链贯穿琵琶骨（视觉冲击）
△他浑身血污，却依旧挺直脊梁（角色特质）

【旁白】（第三人称视角，快速交代）
天庭斩仙台，专门处决触犯天条的仙神。
今日，方寸山最后一个弟子将在此魂飞魄散。（建立危机感）

※慧觉登场（冲突引入）
△金色莲台缓缓落下，慧觉宝相庄严（视觉描写）
【慧觉】（慈悲为怀，眼神冰冷）：林玄，事到如今，你可知罪？（对话设计）
【林玄】（抬头，血污中眼神锐利）：我杀的是披着仙官外衣的畜生！（个性对话）

【卡黑】（悬念设置）
```
