# 剧情拆解示例

## 示例说明
以下是《我的治愈系游戏》前12章的标准剧情拆解示例，展示了如何从小说文本中提取核心冲突点，识别情绪钩子，并进行合理的分集规划。

---

小说名称：《我的治愈系游戏》
小说类型：悬疑/推理/科幻

第1批（第1-6章）

【剧情1】出租屋，韩非戴上《完美人生》头盔被不明物体刺入大脑，危机悬念，第1集，状态：已用
【剧情2】游戏内自家客厅，韩非苏醒发现环境破旧诡异，反差悬念，第1集，状态：已用
【剧情3】楼道，邻居孟诗阿婆邀请韩非吃饺子，温情陷阱，第1集，状态：已用
【剧情4】孟诗家，韩非帮阿婆更换保险丝并激活系统面板，收获爽点，第2集，状态：已用
【剧情5】孟诗家，孙子晨晨摔碗尖叫"我才不吃从棺材里取出的东西"，惊悚反转，第2集，状态：已用
【剧情6】孟诗家厨房，韩非检查冰箱发现黑色塑料袋但无尸体，悬疑铺垫，第3集，状态：已用
【剧情7】孟诗家，孟诗警告韩非"晚上必须锁好卫生间门"，规则悬念，第3集，状态：已用
【剧情8】自家卧室，韩非尝试退出游戏发现被锁死（需玩满3小时），绝望困境，第3集，状态：已用
【剧情9】自家客厅，卫生间门把手自动转动，恐怖降临，第4集，状态：已用
【剧情10】自家客厅，韩非发现沙发旁蹲着一个人影，惊悚高潮，第4集，状态：已用
【剧情11】自家客厅，韩非冲出房门逃生发现楼道灯全灭，生死时速，第4集，状态：已用
【剧情12】楼道，韩非发现整栋楼都贴满符纸且大门上锁，绝境升级，第5集，状态：已用
【剧情13】一楼大门，韩非被鬼勒死前最后一秒退出游戏，极限逃生，第5集，状态：已用
【剧情14】现实自家，韩非发现头盔带血且脑中出现系统警告"泄密即抹杀"，打破次元壁，第6集，状态：已用
【剧情15】现实自家，警察上门询问纵火案，韩非试图求助被系统禁言，无助困境，第6集，状态：已用

第2批（第7-12章）

【剧情16】派出所，韩非接受审讯并结识刑警厉雪，得知古董店老板死于三天前，灵异悬念，第7集，状态：已用
【剧情17】派出所，系统评定韩非危险指数为0，刑警队长反怀疑其为高智商罪犯，反差人设，第7集，状态：已用
【剧情18】《恶之花》片场，韩非试镜受害者魏有福，调动游戏恐惧记忆惊艳全场，打脸爽点，第8集，状态：已用
【剧情19】现实老城区，韩非与厉雪探访孟诗住处，得知孟诗祖孙死于十年前冰柜藏尸案，悬疑揭秘，第9集，状态：已用
【剧情20】现实魏有福家，韩非探访受害者父亲并决定"打不过就加入"（和鬼做朋友），策略转变，第9集，状态：已用
【剧情21】自家卧室，韩非查阅"人体拼图案"资料并于零点再次登录游戏，危机前奏，第10集，状态：已用
【剧情22】游戏内楼道，韩非敲孟诗门无果，与6楼下行的鬼脸遭遇，生死竞速，第10集，状态：已用

## 拆解分析说明

### 冲突提取原则示例
- **剧情1**：头盔刺入大脑 → 明确的生命危机冲突
- **剧情5**：晨晨拒食"死人饭" → 食物安全vs饥饿冲突
- **剧情8**：无法退出游戏 → 自由意志vs强制约束冲突
- **剧情18**：试镜表现惊艳 → 能力质疑vs实力证明冲突

### 情绪钩子识别示例
- **危机悬念**：剧情1（头盔危险）、剧情9（鬼怪出现）、剧情22（生死追逐）
- **温情陷阱**：剧情3（善良邻居实则危险）
- **惊悚反转**：剧情5（孩子拒食揭示真相）、剧情14（游戏影响现实）
- **打脸爽点**：剧情18（被轻视后证明实力）

### 压缩策略示例
**删减内容**：
- 韩非的职业背景详述 → 压缩为"被辞退的演员"
- 长篇心理活动描写 → 提取为简短独白
- 详细环境铺垫 → 保留关键氛围描述

**保留内容**：
- 头盔刺入大脑的惊悚细节 → 核心冲突点
- 孩子的恐惧反应 → 关键线索提示
- 系统面板的具体内容 → 推进剧情的关键信息

### 分集节奏示例
- **第1集**：3个剧情点，从进入游戏到遇见邻居，建立基础设定
- **第4集**：3个剧情点，恐怖高潮集中爆发，紧张度最高
- **第6集**：2个剧情点，现实与游戏交融，转折点设置

这个示例展示了标准的拆解流程和质量要求，为实际改编工作提供参考基准。
