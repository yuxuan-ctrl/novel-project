# 漫剧剧本写作风格规范

## 核心写作理念

### 视觉优先原则
漫剧是视觉媒介，文字描述必须能够直观转化为画面。所有场景、动作、表情都要用具象化的语言描述，避免抽象概念和内心活动。

### 快节奏推进
每一句话都要推动剧情发展，删除一切不必要的修饰和铺垫。保持信息密度高，节奏紧凑。

## 格式符号规范

### 场景标记 `※`
- 用于标记场景转换和环境设定
- 格式：`※ 场景位置，时间，氛围描述`
- 示例：`※ 破旧客厅，深夜，阴暗压抑`

### 视觉描述 `△`
- 用于描述画面、动作、表情、环境细节
- 每个视觉元素单独一行
- 强调视觉冲击力和画面感
- 示例：`△ 韩非瞳孔骤缩，冷汗顺着脸颊滑落`

### 对话标记 `【】`
- 人物对话用【角色名】标记
- 独白用【独白】标记
- 旁白解说用【旁白】标记
- 系统界面用【系统面板】标记

### 特效描述 `【特效】`
- 用于描述视觉特效、音效、氛围效果
- 示例：`【音效】刺耳的金属摩擦声响起`

### 分集结尾 `【卡黑】`
- 每集结尾必须使用此标记
- 表示本集结束，画面切黑

## 语言风格要求

### 动词使用规范
1. **具象动词优先**：用"撞击、撕裂、爆发"替代"影响、产生、出现"
2. **强力动词优先**：用"猛地、狠狠、疯狂"强化动作力度
3. **瞬时动词优先**：用"瞬间、突然、猛然"营造紧张感

### 形容词使用规范
1. **视觉形容词优先**：颜色、形状、大小、材质等可见特征
2. **感官形容词**：触觉、听觉、嗅觉等身体感受
3. **避免情感形容词**：少用"开心、难过、愤怒"等主观情绪词

### 句式结构要求
1. **短句优先**：单句长度控制在20字以内
2. **并列结构**：用并列句式增强节奏感
3. **倒装强调**：重要信息前置，增强冲击力

## 描写技巧规范

### 人物描写
- **外貌特征**：突出能体现性格的特征
- **动作习惯**：通过肢体语言表现内心
- **表情细节**：眼神、嘴角、眉毛等微表情
- 避免：内心独白、性格分析、背景介绍

### 环境描写
- **氛围营造**：光影、声音、气味等感官元素
- **细节特写**：能推动剧情的环境细节
- **对比反差**：用环境反衬情绪或状态
- 避免：大篇幅风景描写、无关环境信息

### 动作描写
- **分解动作**：将复杂动作拆分成连续画面
- **力度描述**：强调动作的速度、力量、幅度
- **结果展现**：动作产生的直观后果
- 避免：抽象动作、心理动作、省略过程

### 对话规范
- **简洁有力**：每句对话都要推动剧情
- **个性化语言**：体现角色特点的说话方式
- **情绪外化**：通过语调、停顿、重音表现情感
- 避免：冗长解释、哲学思辨、无意义闲聊

## 节奏控制技巧

### 紧张节奏营造
1. **短句密集**：用连续短句制造紧迫感
2. **动词堆叠**：连续动作描写加速节奏
3. **感官冲击**：声音、光影、痛觉等刺激

### 悬念节奏控制
1. **信息延迟**：关键信息分段透露
2. **视角切换**：通过不同角度增加悬念
3. **时间停顿**：关键时刻拉长描写

### 爽点节奏把握
1. **蓄势描写**：反转前的铺垫和压抑
2. **瞬间爆发**：爽点时刻的集中释放
3. **余韵回味**：爽点后的震撼效果

## 禁忌内容

### 严禁使用的描写
1. **心理描写**：内心想法、情感分析、思维过程
2. **抽象概念**：哲学思考、概念解释、理论分析
3. **冗长铺垫**：无关紧要的背景、历史、设定
4. **说教内容**：道德评判、价值观输出、教育意图

### 严禁使用的句式
1. **长难句**：超过30字的复杂句式
2. **被动句**：尽量使用主动语态
3. **设问句**：避免自问自答的叙述方式
4. **插叙倒叙**：保持时间线性推进

### 语言禁忌
1. **文言文词汇**：使用现代白话文
2. **网络流行语**：避免时效性强的网络用语
3. **地方方言**：使用标准普通话表达
4. **专业术语**：避免艰深的专业词汇

## 质量检查清单

### 视觉化检查
- [ ] 每个场景都能清晰想象出画面
- [ ] 人物动作具体可操作
- [ ] 环境细节有助于氛围营造
- [ ] 特效描述生动震撼

### 节奏感检查
- [ ] 信息密度适中，不拖沓不跳跃
- [ ] 紧张场面节奏加快
- [ ] 关键时刻重点突出
- [ ] 整体推进流畅自然

### 语言规范检查
- [ ] 符号使用正确规范
- [ ] 句式简洁有力
- [ ] 动词具象生动
- [ ] 形容词恰当精准

### 内容完整性检查
- [ ] 剧情逻辑清晰完整
- [ ] 人物行为合理一致
- [ ] 对话推动情节发展
- [ ] 每集以【卡黑】结尾

遵循这套写作风格规范，确保生成的剧本符合漫剧的视觉特点和节奏要求，为后续的分镜和制作提供优质的文本基础。
