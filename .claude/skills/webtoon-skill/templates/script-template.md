# 单集剧本模板

## 文件命名规范
```
Episode-XX.md （XX为两位数集数，如Episode-01.md）
```

## 集标题格式
```
第X集：集标题（简洁概括本集核心冲突）
```

## 场景标记格式
```
※ 场景位置，时间描述，氛围特征

△ 具体的视觉描述内容
△ 人物动作和表情描述
△ 环境细节和氛围营造
```

## 对话格式
```
△ 角色名（情绪状态/说话方式）："对话内容"
```

## 独白格式
```
【独白】内心想法或旁白内容
```

## 特效格式
```
【音效】声音效果描述
【特效】视觉特效描述
【系统面板】游戏界面或系统提示内容
```

## 结尾标记
每集结尾必须使用：
```
【卡黑】
```

## 完整结构示例

```
第1集：治愈系游戏？

※ 出租屋，深夜，暴雨

△ 窗外电闪雷鸣，破旧窗帘被狂风卷起，像巨大的黑翅膀拍打。
△ 暴雨疯狂砸击玻璃，发出密集的噼啪声。
△ 桌上放着漆黑游戏头盔，表面血红编号：0000。
△ 韩非（25岁，苍白疲惫）盯着头盔，手指颤抖。

【独白】这是我最后的机会了...哪怕是假的也好。

△ 韩非深吸气，戴上头盔。
△ 咔哒一声，头盔严丝合缝扣住。
△ 指示灯亮起刺眼血红光芒！

【音效】刺耳电流声炸响，如指甲刮黑板。

△ 韩非身体猛僵，瞳孔放大。
△ 头盔内十几根尖刺弹出，闪烁寒光！
△ 噗嗤！尖刺扎入后脑、太阳穴！
△ 鲜血涌出，顺脸颊滑落。

---

※ 游戏内，破旧客厅，阴暗

△ 韩非猛弹起，摔下椅子，捂住后脑。
△ 环顾四周，瞳孔骤缩。
△ 破旧公寓，墙皮脱落，露出发霉砖块。

【独白】这就是...治愈系游戏？

【卡黑】
```

## 写作规范要点

### 字数控制
- 单集总字数：500-800字
- 场景描述：简洁有力，突出关键信息
- 对话内容：推动剧情，体现角色特点

### 节奏控制
- **开头**：立即进入冲突，抓住注意力
- **中段**：稳定推进，保持张力
- **结尾**：设置悬念，引导下集

### 视觉描述要求
- 使用△符号标记每个视觉元素
- 描述具体可画的画面内容
- 突出动作的力度和速度
- 强化环境氛围营造

### 对话写作要求
- 简洁有力，避免冗长
- 符合角色性格特点
- 推动剧情发展
- 体现情绪变化

## 质量检查要点

### 格式检查
- [ ] 符号使用正确（※△【】）
- [ ] 场景标记完整
- [ ] 对话格式规范
- [ ] 结尾【卡黑】标记

### 内容检查
- [ ] 剧情点完整还原
- [ ] 字数控制在范围内
- [ ] 节奏紧凑不拖沓
- [ ] 视觉描述生动

### 逻辑检查
- [ ] 情节推进合理
- [ ] 人物行为一致
- [ ] 前后呼应清晰
- [ ] 悬念设置有效

## 注意事项

1. **严格按模板格式**：不得随意更改符号和结构
2. **视觉化优先**：所有描述都要能转化为画面
3. **节奏感控制**：紧张时短句密集，舒缓时适当延长
4. **情绪钩子突出**：重点描写能引发情绪反应的情节
5. **集间连贯**：考虑与前后集的逻辑连接
6. **悬念设置**：每集结尾留下引人入胜的钩子

遵循此模板创作的剧本将具备良好的视觉效果和节奏感，适合改编为漫剧内容。
